**Tracer** is inherited by any other tracer modules.

## Methods

| Method | Description |
|--------|-------------|
| **Tracer**((String) name)| create Tracer and set its name |
| **_setData**(args) | set data to visualize |
| **_clear**() | clear the tracer |
| **_wait**() | wait for a certain amount of time |
